Advanced Project Submissions

Project 1: weVOLT Core Navigation Flow

Problem Statement: The adoption of electric vehicles (EVs) is increasing, but many EV users still face difficulties in locating available charging stations, checking charger availability, and planning their charging activities efficiently. Existing solutions may provide limited real-time information, leading to inconvenience, longer waiting times, and range anxiety among drivers. Therefore, WeVolt was developed to provide a centralized platform that helps users easily find charging stations, access relevant charging information, and manage their EV charging experience more effectively.

Lessons Learned Working on WeVolt taught me that building a system is not just about writing code, but also about understanding real user problems and designing with those needs in mind. I realised how easy it is to underestimate small issues like data flow, edge cases, and user experience until I actually faced them during development. This project also helped me become more patient and consistent, especially when things didn’t work as expected and I had to keep debugging and improving step by step.

Project 2: weVOLT Asynchronous Ecosystem

Problem Statement: WeVolt integrates the Open Charge Map API to ensure users have access to accurate and up-to-date EV charging station information, as maintaining a manual database would be inefficient and quickly outdated. QR code scanning is implemented to simplify user interaction at charging stations and reduce errors during identification. Firebase Firestore is used to enable real-time data storage and synchronization across users and devices.

Lessons Learned Through this project, I learned how important it is to combine different technologies to build a complete and functional system rather than relying on a single approach. I also gained a better understanding of how APIs, hardware interaction (QR scanning), and cloud databases work together to improve system efficiency. Most importantly, I learned how these integrations directly impact user experience and system reliability.
